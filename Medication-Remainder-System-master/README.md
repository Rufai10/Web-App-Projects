# React + Vite
# 💊 Medication Reminder System

This is a web-based **Medication Reminder System** built with **React.js** using **Vite**. The system allows users to manage medications, schedule reminders, and monitor medication history through a user-friendly interface. It supports admin and user roles with protected routes and calendar features.

---

## 🌐 Live Site

🔗 [View Project on GitHub Pages](https://muhammedOsman11.github.io/Medication-Remainder-System/)

---

## 📂 Features

- ✅ User & Admin Authentication
- 💬 Medication scheduling and notifications
- 🗓 Medication calendar and dashboard
- 📋 Medication management (Add/Edit/Delete)
- 👨‍⚕️ Admin panel to manage users and system settings
- 🔐 Protected Routes
- 📱 Responsive UI with TailwindCSS
- 🎨 Light/clean tech theme design

## 🛠 Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/muhammedOsman11/Medication-Remainder-System.git
   cd Medication-Remainder-System


2.Install Dependencies
npm install

3.Run the App (Development)
npm run dev


4.Build for Production
npm run build

5.Deploy to GitHub Pages
npm run deploy
