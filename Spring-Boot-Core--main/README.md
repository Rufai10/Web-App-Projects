## 📌 Project Overview

This project is a Spring Boot application developed as part of a core concepts exercise. It demonstrates the creation of a RESTful API with layered architecture, CRUD operations, and testing using Postman and Insomnia.

---

## ⚙️ Technologies Used

- **Java 24**
- **Spring Boot 3.x**
- **Lombok**
- **Postman / Insomnia**
- **IntelliJ IDEA**
